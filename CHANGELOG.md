## [0.1.0] — 2025-05-11

### Added
- Initial project setup (WPF, .NET 8)
- README with project description and privacy manifesto
- 10 planned conversion types listed as Good First Issues
  - PNG / JPG / WEBP ↔ each other
  - PDF → DOCX
  - DOCX / TXT → PDF
  - MP4 → MP3
  - CSV → XLSX
  - SVG → PNG / PDF
  - MP3 / WAV ↔ each other
  - HEIC → JPG / PNG
  - Markdown → HTML / PDF
  - ZIP / RAR ↔ ZIP / 7Z
